User Stories
============
User Story
	Hypothesises (assumptions)
	Has a Name
	Has a Description (either in Gherkin or not)
	Is a placeholder for further conversation
	Deliver incremental value

A user story has the same exact format as a Value Story, only that it's focused solely on a personal, lower level executable feature / value. A value story can also focus on different levels such as persona, business, or technical but they're more general value statements (OKRs)

	As a (an) [persona]
	I want [feature/goal]
	So that [benefit/outcome]
